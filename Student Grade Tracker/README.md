# Student Grade Tracker

This is a simple web application to calculate student
total marks, average, and result status.

## What this project does
- Takes student name and subject marks as input
- Calculates total marks and average
- Shows pass or fail result
- Displays data in a table
- Allows clearing all data

## Technologies used
- HTML
- CSS
- JavaScript

## JavaScript concepts used
- Variables and functions
- Event handling
- Conditional statements
- DOM manipulation

## Purpose
This project is created for practicing basic
JavaScript and form handling.
